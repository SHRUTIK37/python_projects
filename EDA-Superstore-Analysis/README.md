### Description
Performed Exploratory Data Analysis (EDA) on the Superstore dataset using Python to uncover sales, profit, discount, and regional trends. The project focuses on data cleaning, visualization, correlation analysis, and outlier detection.

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

### Key Analysis Performed
- Data inspection and preprocessing
- Data type conversion for identifiers
- Bar charts for categorical analysis
- Scatter plots for Sales vs Profit and Discount vs Profit
- Histograms for distribution analysis
- Correlation heatmaps
- Outlier detection using boxplots

### Key Insights
- Identified relationships between sales, profit, discount, and quantity
- Analyzed category-wise, region-wise, and segment-wise distributions
- Detected outliers impacting sales and profit
